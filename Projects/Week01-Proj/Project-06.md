# Projects For Week 1 (10 days)

## Project 6: JavaScript Todo List 1

**What You Will Build:**

- A fully functional Todo List using HTML, CSS, and JavaScript
- Add and delete tasks dynamically
- Practice DOM manipulation, event handling, and array operations
- Responsive and user-friendly interface

**Key Concepts Covered:**
- JavaScript arrays and objects
- DOM manipulation (createElement, appendChild, removeChild)
- Event listeners (add, delete)
- Input validation
- Local storage (optional)
- CSS for layout and design
