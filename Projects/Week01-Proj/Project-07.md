# Projects For Week 1 (10 days)

## Project 7: JavaScript Todo List 2

**What You Will Build:**

- An advanced Todo List using HTML, CSS, and JavaScript
- Add, edit, delete tasks dynamically
- Mark tasks as completed
- Practice DOM manipulation, event handling, and array operations
- Responsive and user-friendly interface

**Key Concepts Covered:**
- JavaScript arrays and objects
- DOM manipulation (createElement, appendChild, removeChild, setAttribute)
- Event listeners (add, edit, delete, complete)
- Input validation
- Local storage (optional)
- CSS for layout and design
