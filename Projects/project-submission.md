# Project Submission Instructions

## 1. Create a GitHub Repository and Push Your Code

```bash
# Create a repo in github and copy it's url
# Initialize git and push to GitHub

git init
git add ./
git commit -m "first commit"
git branch -M main
git remote add origin <github-repo-url>
git push -u origin main
```

---

## 2. Deployment

### CSS/JS Projects (GitHub Pages)
- Deploy your static site using GitHub Pages:
	1. Go to your repository on GitHub.
	2. Click on 'Settings' > 'Pages'.
	3. Select the branch (usually `main`) and `/root` or `/docs` folder.
	4. Save and get your live site URL.
- **Demo Video URL:** [Link](https://youtu.be/BT4WzyT2g8k?si=lJya-rhafQkeF2CE)

---

### React Projects (Vercel)
- Deploy your React app using Vercel:
	1. Go to [Vercel](https://vercel.com/) and sign in with GitHub.
	2. Import your repository and follow the prompts to deploy.
	3. Copy your deployed site URL.
